UGen.UGenInput

getIncomingUGen

Description

The incoming UGen is the UGen that is patched to this UGenInput. When this input is ticked, it will tick the incoming UGen and store the result in its last values.

Signature

UGen getIncomingUGen()

Returns

the UGen that is patched to this UGenInput
